#### Liked the git hooks here?
If you want to use these git hooks for your python projects here are the
instructions:
1. install pre-commit:
    ```bash
        pip install pre-commit
    ```
2. run 
    ```bash
        pre-commit install
    ```
3. Code Away!!!
